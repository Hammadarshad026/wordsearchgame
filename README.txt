# 🧩 Word Search Game - Java Console Application

## 🎮 Description

This is a command-line based **Word Search Game** developed in **Java**, using core data structures and algorithms. The player must find hidden words on a randomly generated board by entering the word along with its coordinates. The game supports **undo**, **redo**, **timed gameplay**, and keeps a **history of found words**.

---

## 🚀 Features

- ✅ 10x10 random word grid
- ✅ Words placed horizontally
- ✅ Validate found words using coordinates
- 🔁 Undo/Redo functionality
- ⏳ 2-minute countdown timer
- 🧠 Game history tracking
- 🧼 Input validation and feedback

---
